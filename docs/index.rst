   :maxdepth: 3

A miscellaneous set of utilities for python3.6+ programs and scripts.

Installation
------------

To install the ccautils package run this command in your console

.. code-block:: console

  $ pip install ccautils --user

Usage
-----

.. code-block:: python

  import ccautils.utils as UT
  import ccautils.fileutils as FT
  from ccautils.errors import errorNotify
  from ccautils.errors import errorRaise
  from ccautils.errors import errorExit
